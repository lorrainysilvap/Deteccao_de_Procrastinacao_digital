# Detecção de procrastinação digital por meio da análise de uso de aplicativos e navegação web

Este projeto tem como objetivo desenvolver uma aplicação para PC capaz de detectar padrões de procrastinação digital a partir da análise do uso de aplicativos e da navegação web.
A ferramenta será voltada principalmente para o meio acadêmico, podendo ser implementada em laboratórios de informática de escolas, universidades e bibliotecas, auxiliando no acompanhamento da produtividade e no incentivo ao uso consciente dos recursos digitais durante atividades de estudo e pesquisa.

<h2 id="objetivo"> 🎯 Objetivo</h2>

* Monitorar o uso do computador (aplicativos, tempo de uso, troca de abas/janelas);
* Identificar padrões de distração e procrastinação digital durante períodos de estudo;
* Auxiliar instituições de ensino no acompanhamento do uso adequado dos laboratórios e bibliotecas;
* Enviar alertas quando o usuário estiver distraído por muito tempo;
* Sugerir pausas ou bloqueios de aplicativos/sites específicos;
* Gerar relatórios e gráficos de produtividade, destacando os principais fatores de distração nos ambientes educacionais.

---

<h2 id="funcionalidades-principais"> 🧠 Funcionalidades Principais</h2>

* Monitoramento em segundo plano do uso de aplicativos;
* Registro de troca de abas/janelas e tempo gasto em cada uma;
* Classificação de aplicativos/sites (produtivos x distrações);
* Alertas em tempo real: “Você está distraído há X minutos”;
* Sugestões de foco (bloqueio temporário, modo focado, pausas programadas);
* Dashboard com gráficos de:
   - Tempo produtivo x tempo de distração;
   - Aplicativos/sites mais usados;
   - Horários de maior procrastinação.

<h2 id="tecnologias-e-ferramentas-utilizadas"> 🛠️ Tecnologias e Ferramentas Utilizadas</h2>

* **Figma** – prototipação de design da interface do usuário (UI/UX) focado em experiência de produtividade.
* **Python** – linguagem principal para a lógica de negócio, processamento de dados e integração com o sistema operacional.
* **Flet** – framework para desenvolvimento da interface gráfica nativa (GUI), permitindo uma experiência desktop moderna e responsiva via Flutter engine.
* **SQLite** – banco de dados leve para armazenamento e persistência do histórico de atividades do usuário.
* **FastAPI** – construção de uma camada de API interna performática para organização de serviços e comunicação entre processos.
* **Psutil & PyGetWindow** – Bibliotecas de baixo nível para monitoramento de processos ativos, títulos de janelas e consumo de recursos do sistema em tempo real.
